# Customer Purchase Behavior Analysis Dashboard (Excel)

##  Project Overview

This project showcases an interactive **Customer Purchase Behavior Analysis Dashboard** built entirely in **Microsoft Excel**. The dashboard transforms raw customer purchase data into meaningful business insights using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

It enables users to analyze customer purchasing patterns, revenue trends, payment preferences, shipping methods, seasonal performance, and demographic insights through an interactive interface.

---

##  Objective

The objective of this project is to analyze customer purchase behavior and create an interactive dashboard that helps businesses make data-driven decisions.

##  Key Business Insights

- Clothing contributes **44.7%** of the total revenue.
- Free Shipping generates the highest purchase amount.
- Adult customers contribute the highest revenue.
- Customers with ratings between **4.0–5.0** represent the largest customer group.
- Fall season slightly outperforms the other seasons in total revenue.
